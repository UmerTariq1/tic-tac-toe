# Tic Tac Toe

A bright, responsive tic-tac-toe game with a timed turn rule and win celebration.

## Features

- **15-second turn timer** — Each player has 15 seconds per turn; running out of time means the other player wins.
- **Distinct colours** — X uses pink/magenta, O uses blue; the timer bar matches the current player.
- **Win highlight** — The winning row/column/diagonal is highlighted; confetti plays when someone wins (by line or by timeout).
- **Restart** — Restart button resets the board and timer.

## How to run

Open `index.html` in a modern browser. No build step or server required.

## Tech

- Vanilla HTML, CSS, and JavaScript.
- Nunito font (Google Fonts).
- No external libraries.
